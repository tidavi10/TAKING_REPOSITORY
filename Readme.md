# Teste Taking Alelo

## Descrição
1) Uma empresa de filmes, contratou um sistema da sua empresa - considerando que existe a necessidade de cadastrar diretor (nome, nascimento) e filmes (nome, ano de lançamento) construa um sistema utilizando arraylist para atender esta necessidade e no final, construa um codigo de automação (utilizar selenium, Junit e chromeDriver que busque no google o diretor e o filme e retorne a quantidade aproximada de resultados para a busca.
2) Construa uma chamada de api em java usando jayway rest assured e valide a mensagem da url:

```link
https://jsonplaceholder.typicode.com/todos/1
```

-Será avaliado:
- Indentação
- Orientação a Objetos
- Logica implementada
- Boas práticas de automação - Tratamento de erros
- Validações

## Para rodar os testes
Usando o Chrome:  
```bash
mvn clean test -Dwebbrowser=chrome
```
